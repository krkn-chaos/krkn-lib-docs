utils.safe_logger module¶

class utils.safe_logger.SafeLogger(filename: str | None = None, write_mode: str | None = None)¶

Bases: object

__init__(filename: str | None = None, write_mode: str | None = None)¶

SafeLogger provides a mechanism to thread-safely log if initialized with a filename and optionally with a write mode (default is w+) otherwise will work as a wrapper around the logging package. The double working mechanism is meant to not force file logging to the methods that depend on it. :param filename: the log file name, if None the class will behave as a simple logger :param write_mode: file write mode, default is w+
